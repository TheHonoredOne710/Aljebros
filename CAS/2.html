Algebra 1 Formulas

The slope of a line represents its steepness and is calculated as:

Formula: m = (y₂ - y₁) / (x₂ - x₁)

Where (x₁, y₁) and (x₂, y₂) are two points on the line. For example, if the points are (1, 2) and (3, 6):

m = (6 - 2) / (3 - 1) = 4 / 2 = 2

The slope-intercept form of a linear equation is written as:

Formula: y = mx + b

Where:

  • m is the slope of the line.
  • b is the y-intercept (the point where the line crosses the y-axis).

Example: If m = 2 and b = 3, then the equation is y = 2x + 3.

The point-slope form of a linear equation is used when the slope m and a point on the line (x₁, y₁) are known:

Formula: y - y₁ = m(x - x₁)

Example: If m = 3 and the point is (2, 4), the equation is:

y - 4 = 3(x - 2) → y = 3x - 6 + 4 → y = 3x - 2

The standard form of a linear equation is written as:

Formula: Ax + By = C

Where:

  • A, B, and C are integers.
  • A and B cannot both be 0.

Example: The equation 3x + 2y = 12 is in standard form.

The quadratic formula is used to find solutions (roots) of a quadratic equation ax² + bx + c = 0:

Formula: x = (-b ± √(b² - 4ac)) / 2a

Example: For the equation 2x² + 4x - 6 = 0, substitute a = 2, b = 4, and c = -6:

x = [ -4 ± √(4² - 4(2)(-6)) ] / 2(2) → x = [-4 ± √(16 + 48)] / 4 → x = [-4 ± √64] / 4 → x = (-4 ± 8) / 4

Solutions: x = 1 and x = -3.

Factoring is used to rewrite a quadratic equation in the form (x + p)(x + q) = 0, making it easier to find the roots:

Example: Factor x² + 5x + 6:

(x + 2)(x + 3) = 0

Roots: x = -2 and x = -3.

The distance formula calculates the distance between two points (x₁, y₁) and (x₂, y₂) in a plane:

Formula: d = √[(x₂ - x₁)² + (y₂ - y₁)²]

Example: For points (1, 2) and (4, 6):

d = √[(4 - 1)² + (6 - 2)²] → d = √[3² + 4²] → d = √(9 + 16) → d = √25 = 5

The midpoint formula finds the point exactly halfway between two points (x₁, y₁) and (x₂, y₂):

Formula: M = [(x₁ + x₂) / 2, (y₁ + y₂) / 2]

Example: For points (2, 3) and (4, 7):

M = [(2 + 4) / 2, (3 + 7) / 2] → M = [6 / 2, 10 / 2] → M = (3, 5)
